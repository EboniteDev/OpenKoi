# OpenKoi
A Koikoi game made with the Godot game engine.
This project was made in order to have a widely distributed application of the Koikoi hanafuda card game available on Linux systems and a better alternative on other platforms like iOS.
## General
  The Godot project was started with the "Compatible" option, this should allow the sideloading/use for mobile devices or on a website.
